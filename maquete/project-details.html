Restaurant-of-Tangier

A dynamic web application showcasing restaurants in Tangier, with features like detailed pages, admin panel, and search.

Project Overview

Restaurant-of-Tangier is a full-stack web application designed to help users discover and explore restaurants in Tangier, Morocco. It provides a seamless user experience with detailed restaurant pages, a search functionality, and an admin panel for managing content.

The application is built using Laravel for the backend, handling data storage and retrieval with MySQL. The frontend is crafted with HTML5, CSS3, and Tailwind CSS for a responsive and modern design. Key features include user authentication, restaurant filtering, and an admin dashboard for CRUD operations.

Technologies Used

HTML5 Laravel MySQL Tailwind CSS

Key Features

Restaurant Listings

Browse detailed pages for restaurants with information on cuisine, location, and ratings.

Search & Filter

Search restaurants by name, cuisine, or location with dynamic filtering options.

Admin Panel

Manage restaurant data with a secure admin dashboard for adding, editing, and deleting entries.

User Authentication

Secure login and registration system for users to access personalized features.

Challenges & Solutions

Challenge: Efficient Data Retrieval

Fetching and displaying restaurant data efficiently for a smooth user experience.

Solution: Implemented Laravel's Eloquent ORM with query optimization and caching to reduce database load.

Challenge: Responsive Design

Ensuring the application looks great and functions well on all devices.

Solution: Used Tailwind CSS for responsive layouts and tested across multiple screen sizes.

Challenge: Secure Admin Access

Preventing unauthorized access to the admin panel.

Solution: Integrated Laravel's middleware for authentication and role-based access control.
